# Terminal Commands
Install NodeJs from NodeJs Official Page.
Open Terminal
Go to your file project
Run in terminal: npm install -g @angular/cli
### npm install

### Run  npm install jquery@3.5.1  to resolve 2 vulnerabilities
### Run  npm install bootstrap@4.5.2  to resolve 1 vulnerability
### Run  npm install --save-dev karma@5.2.3  to resolve 1 vulnerability
### Run  npm install --save-dev protractor@7.0.0  to resolve 1 vulnerability
### Run  npm install --save-dev @angular-devkit/build-angular@0.1001.3  to resolve 3 vulnerabilities
### Run  npm install --save-dev protractor@7.0.0  to resolve 1 vulnerability

### npm start
### http://localhost:4200/

### ng build --prod --base-href "http://georgehowell.com.au/GH-2020/"
